# Simple games written in Python

These games were created to learn the [pygame](https://www.pygame.org/) module, and exercise the basics of game development.

It is also possible that in the future an automatic machine learning based solver will be developed for these games.

The reconstructed games include the:
- [Laptop mini-game from nopixel](https://nopixel.fandom.com/wiki/Heist_Hacks)
- [Thermite mini-game from nopixel](https://nopixel.fandom.com/wiki/Heist_Hacks)
- [Tetris game](https://tetris.com/play-tetris)


For detailed information about how the games work take a look at the games' folder.

## Installation

To run the games you only need the `pygame` and `numpy` package installed besides `python`.

Create a new virtual environment:
`python -m venv .venv`

Activate the environment (Windows):
`.venv\Scripts\activate.bat`

Install the packages:
`pip install -r requirements.txt`

And you are ready to run the game:
`python laptop/game/laptop.py`
